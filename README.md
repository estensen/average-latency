# ping
```
NAME:
   ping - get latencies of web sites

USAGE:
   ping [global options] command [command options] [arguments...]

VERSION:
   0.0.0

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --average, -a  Display average latency (default: false)
   --help, -h     show help (default: false)
   --version, -v  print the version (default: false)
```
